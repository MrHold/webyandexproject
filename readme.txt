Проект для Лицея Академии Яндекса: Веб-приложение на Flask - Проверка слова.
Мой проект реализован с использованием Flask, flask-wtf и прочих дополнений Flask, pymorphy3 для анализа слова, SQLAlchemy, в роли ORM и других библиотек. (полный список в requirements.txt)
При запуске файла main.py сайт можно открыть просто введя в пойсковую строку localhost:5000.
На главной странице сайта есть поле для ввода искомого слова, значок для смены темы сайта (солнце/луна), вкладка «О проекте» и вкладка авторизации.
Поиск выполняется API Яндекс.Спеллер. Это быстро, эффективно, не нужно изобретать велосипед, ну и сервис пока не падал.
При поиске какого-либо слова, выводится его правильная версия, или если она не была найдена оно само, его начальная форма, вид, род (я назвал это «граммемы»).
Если слово ищет авторизованный пользователь, оно добавляется в список слов, которые он искал. Этот список можно увидеть нажав на свой никнейм справа сверху шапки сайта.
При авторизации, если пользователь не зарегистрирован, он может нажать снизу ссылку «Регистрация». Также присутствует защита от ботов — «Капча».
Кроме просмотра слов, в профиле можно поиграть в игру по словам. При вводе слова на главной, если оно неправильно написано и при проверке была найдена его правильная версия, оно записывается в базу данных слов для игры. У пользователя бесконечное количество попыток, при уходе со страницы на другие, игра начнётся сначала.
